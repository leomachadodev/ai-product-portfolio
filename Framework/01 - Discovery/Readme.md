# Discovery

## Objetivo

Descobrir se existe um problema real antes de construir qualquer solução.

---

## Entradas

- Ideia
- Problema observado
- Necessidade de mercado

---

## Perguntas

### Problema

- Qual problema existe?
- Como ele foi identificado?
- Qual seu impacto?

### Usuário

- Quem sofre com esse problema?
- Quem toma a decisão?
- Quem utilizará o produto?

### Mercado

- Existe demanda?
- Existem concorrentes?
- Como resolvem hoje?

### Oportunidade

- O que pode ser feito diferente?
- Por que vale construir esse produto?

---

## Artefatos

- Problema documentado
- Pesquisa inicial
- Hipóteses
- Concorrentes
- Insights

---
