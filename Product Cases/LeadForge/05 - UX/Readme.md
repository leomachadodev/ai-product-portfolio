# LeadForge

# 05 - UX

## Visão Geral

A experiência do usuário foi projetada para reduzir a complexidade do processo de prospecção comercial, organizando toda a operação em uma sequência lógica de etapas.

Cada CRM representa um nicho de mercado independente, permitindo que o usuário configure sua estratégia comercial, Inteligência Artificial, serviços e pipeline antes mesmo de iniciar a busca por novos clientes.

A navegação foi concebida para minimizar tarefas repetitivas e concentrar todas as informações necessárias em um único ambiente.

---

# Princípios de UX

Durante a concepção do produto foram considerados os seguintes princípios:

- Fluxo linear de prospecção.
- Redução de tarefas operacionais.
- Organização por nicho de mercado.
- Navegação simples.
- Centralização das informações.
- Facilidade de aprendizado.
