# ScoutIQ

# 05 - UX

## Visão Geral

A experiência do usuário do ScoutIQ foi projetada para tornar o acompanhamento do desenvolvimento esportivo simples, intuitivo e orientado por dados.

A plataforma foi concebida para atender diferentes perfis de usuários — atletas, pais, clubes, treinadores e scouts — oferecendo uma navegação organizada de acordo com as necessidades de cada um.

Toda a experiência foi construída para reduzir o tempo gasto com registros manuais, facilitar a consulta das informações e transformar dados esportivos em conhecimento para apoio às decisões.

---

# Personas

## Persona Principal

### Atleta

Objetivos

- Acompanhar sua evolução esportiva.
- Construir um histórico organizado.
- Compartilhar seu perfil com clubes.
- Identificar pontos de melhoria.

Necessidades

- Facilidade de utilização.
- Indicadores claros.
- Evolução visual.
- Perfil profissional.

### Pais e Responsáveis

Objetivos

- Registrar partidas.
- Acompanhar a evolução do atleta.
- Organizar o histórico esportivo.
- Compartilhar informações com clubes.

## Persona Terciária

### Clubes e Escolinhas

Objetivos

- Gerenciar atletas.
- Organizar categorias.

## Persona Quaternária

### Scouts e Empresários

Objetivos

- Consultar perfis esportivos.
- Apoiar processos de avaliação.

# Princípios de UX

Durante a concepção da plataforma foram adotados os seguintes princípios.

- Simplicidade de utilização.
- Baixa curva de aprendizado.
- Navegação intuitiva.
- Centralização das informações.
- Feedback visual imediato.
- Organização cronológica da evolução do atleta.
- Informações apresentadas de forma clara e objetiva.

# Responsividade

A plataforma foi projetada para utilização em:

- Desktop
- Tablet
- Smartphone

O registro dos eventos da partida foi otimizado para dispositivos móveis, permitindo que pais, responsáveis ou avaliadores realizem os lançamentos de forma rápida durante o acompanhamento do jogo.

# Acessibilidade

Boas práticas consideradas durante o desenvolvimento da experiência.

- Hierarquia visual consistente.
- Componentes reutilizáveis.
- Feedback visual para ações do usuário.
- Navegação padronizada.
- Contraste adequado.
- Organização lógica das informações.

# Resultado da UX

Ao concluir esta etapa foi definida toda a experiência do usuário do ScoutIQ, estabelecendo jornadas, fluxos de navegação, arquitetura da informação e organização das principais interfaces da plataforma, garantindo uma experiência consistente para atletas, famílias, clubes e avaliadores durante toda a jornada de desenvolvimento esportivo.
